# Resume Builder

A modern, dynamic web application that allows users to create, customize, and export professional resumes. Built with a focus on aesthetics and user experience, this tool provides real-time previewing and easy PDF export.

## Features

- **Real-time Preview**: See your changes instantly as you type.
- **Direct Editing**: Click directly on the resume preview to make quick edits.
- **Customization**:
  - Change accent colors.
  - Adjust font sizes.
- **Data Management**:
  - Save your progress locally (auto-save).
  - Export your data as a JSON file.
  - Import previously saved JSON data.
- **PDF Export**: Download your resume as a clean, professionally formatted PDF.
- **Sections**:
  - Personal Information
  - Professional Summary
  - Education
  - Experience
  - Projects
  - Certifications
  - Languages

## Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Backend**: Node.js, Express (for serving the app)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/YourUsername/resume-builder.git
    cd resume-builder
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the application:
    ```bash
    npm start
    ```

4.  Open your browser and visit:
    ```
    http://localhost:3000
    ```

## Usage

1.  **Fill in your details**: Use the form on the left to enter your resume information.
2.  **Customize**: Use the "Design & Layout" section to pick your preferred colors and font size.
3.  **Review**: Check the live preview on the right. You can also click on the text in the preview to edit it directly.
4.  **Save/Load**: Use the "Save / Export JSON" button to keep a backup of your data.
5.  **Download**: Click "Download PDF" to generate the final file.
